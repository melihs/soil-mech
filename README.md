# Soil bearing capacity calculation application

An application to calculate floor bearing capacity using the terzaghi formula

  

## Getting Started
  

### Demo

https://soil-mech.herokuapp.com/

  

### Run
  
```

$ git clone https://github.com/melihs/soil-mech.git

$ cd soil-mech

$ npm run start

```
Open the browser

```
http://localhost:3000

```
